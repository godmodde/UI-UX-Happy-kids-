# 04 · UI System — базовая дизайн-система (v1.0)

Статус: ✅ Утверждено · Зависит от: [03-PAGE-MAP](03-PAGE-MAP.md)
Бренд: **Happy Kids**. Принцип: игрово, но со зрительным комфортом (не пересыщено), «tween 12+».

## Цвет — комплементарная схема (оранжевый × синий)
Тёплый оранжевый = основной (бренд/CTA), комплемент azure-синий = доверие/«родительский».
Между ними — нейтральная тёплая база для зрительного комфорта.
| Токен | HEX | Назначение |
|---|---|---|
| `--bg` | `#FFF8F2` | Тёплый off-white фон (персиковый подтон = комфорт) |
| `--surface` | `#FFFFFF` | Карточки |
| `--ink` | `#22252D` | Текст (не чёрный — мягче) |
| `--ink-muted` | `#6B7280` | Вторичный текст |
| `--primary` | `#FF7A33` | Бренд/CTA — тёплый оранжевый (база схемы) |
| `--primary-600` | `#E85F1A` | Hover/active primary |
| `--primary-tint` | `#FFE7D6` | Подложки, мягкие зоны |
| `--complement` | `#2E7DF6` | Комплемент (azure-синий): доверие, ссылки, чекаут |
| `--complement-tint` | `#DCEBFF` | Подложки «спокойных» блоков (корзина/чекаут) |
| `--accent-warm` | `#FFC24D` | Аналог-жёлтый для бейджей/иллюстраций (тёплая ветка) |
| `--success` | `#22C55E` | Успех / бейдж «Новинка» |
Подложки карточек товара — мягкие тинты из тёплой ветки (персик/жёлтый) + редкий голубой.
Контраст текста — **WCAG AA** (зрительный комфорт = требование клиента): `--ink` на `--bg`,
белый текст на `--primary-600` для CTA (на `#FF7A33` тёмный текст для AA).

## Типографика
Обе гарнитуры — **SIL Open Font License (OFL)**: свободны, в т.ч. для коммерции, без оплаты.
- **Заголовки/дисплей: Fredoka** (округлый, дружелюбный, не инфантильный) — H1–H3, бейджи, цены.
- **Содержание/UI: Nunito Sans** (OFL) — мягкий гуманистический гротеск, скруглённые формы
  рифмуются с Fredoka, отлично читается в body на RU и EN. Резервный системный стек:
  `"Nunito Sans", system-ui, -apple-system, Segoe UI, Roboto, sans-serif`.
- (Geist в репо оставляем только под технический/моно-текст при необходимости — не основной.)
- Шкала (desktop): H1 48 / H2 32 / H3 24 / body 16 / small 14. Mobile −1 ступень.
- Вес: Fredoka 500/600 для заголовков; Nunito Sans 400 body, 600 для акцентов/кнопок.

## Сетка и пространство
- 12-колоночная сетка, контейнер 1200px, gutter 24px.
- Spacing-шкала (8pt): 4 · 8 · 12 · 16 · 24 · 32 · 48 · 64.
- Радиусы: `--r-sm 8` · `--r-md 16` · `--r-lg 24` · карточки/кнопки крупные и скруглённые.
- Тени мягкие, рассеянные (без жёстких краёв) — спокойствие для глаз.

## Компоненты (обязательный набор из брифа)
- **Карточка товара**: фото на пастельной подложке, название, серия, цена (₽/$), бейджи, кнопка «В корзину».
- **Фильтры**: чипы/аккордеоны — возраст, детали (≤100), серия, сложность, цена (слайдер), бренд.
- **Навигация**: шапка (лого, Каталог, поиск, язык RU/EN, валюта ₽/$, корзина) + футер.
- **CTA-кнопки**: primary (заливка `--primary`), secondary (контур), большой тап-таргет (≥44px — равный приоритет устройств).
- **Модальные окна**: быстрый просмотр товара, мини-корзина.
- **Формы**: чекаут/контакты — крупные поля, понятные ошибки (стиль «для родителя»).
- **Бейджи**: Новинка, Скидка, Сложность (простая), Возраст 12+, Кол-во деталей.

## Состояния и доступность
- Состояния для всех интерактивов: default/hover/focus/active/disabled, видимый focus-ring.
- Адаптив: брейкпоинты 360 / 768 / 1024 / 1280. Карточки 2 кол. (mobile) → 3–4 (desktop).
- Анимации: лёгкие, ненавязчивые (микро-hover карточек), уважать `prefers-reduced-motion`.

## История по заголовочному шрифту (зафиксировано)
Изначально выбрали **Fredoka**, но проверка субсетов в `next/font` показала: Fredoka
покрывает только `latin / latin-ext / hebrew` — **кириллицы нет**, RU-заголовки не работают.
Из шрифтов с кириллицей подобрана ближайшая замена по характеру (округлый геометрический):
**Comfortaa** (✅ cyrillic, OFL). Альтернативы рассмотрены: Unbounded, Rubik (обе с кириллицей).
Итог: **Comfortaa** — единый заголовочный шрифт для RU и EN.

## Решено (v1.0)
1. Бренд — **Happy Kids**.
2. Палитра — **комплементарная**: оранжевый `#FF7A33` × синий `#2E7DF6`.
3. Заголовки — **Comfortaa** (OFL, с кириллицей; замена Fredoka из-за отсутствия RU).
4. Содержание — **Nunito Sans** (OFL). Пара Comfortaa + Nunito Sans, обе свободные, обе с кириллицей.
