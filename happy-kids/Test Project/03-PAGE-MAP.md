# 03 · Page Map — карта страниц

Статус: ✅ · Зависит от: [02-ANALYSIS](02-ANALYSIS.md)

Обязательная e-commerce структура из брифа и решение по каждому пункту (каталог < 50):

| Страница | Включаем | Решение / комментарий |
|---|---|---|
| **Home** (витрина) | ✅ | Hero под ребёнка + блок серий + бестселлеры + доверие для родителя |
| **Category / Collection** | ✅ | Одна страница каталога с фильтрами (плоская структура) |
| **Product (PDP)** | ✅ | Галерея, переключатель **серий**, цена/валюта, бейджи, «В корзину» |
| **Cart** | ✅ | Спокойный «родительский» стиль, правки кол-ва, промокод |
| **Checkout** | ✅ | **Гостевой** чекаут в 1 экран/шаги: контакты → доставка → оплата |
| **Search** | ⚠️ упрощён | Строка в шапке + результаты на странице каталога (мало SKU) |
| **Filters system** | ✅ | Возраст, кол-во деталей (≤100), серия, сложность, цена, бренд |
| **Account** | ❌ опц. (фаза 2) | **Бизнес-решение:** на MVP не делаем — родителю важен гостевой чекаут |

Доп. страницы: `Checkout Success`, `404`, статичные (Доставка/Оплата, О бренде — доверие родителя).

## Карта переходов
```
Home ─┬─> Category (filters) ─> PDP ─> Cart ─> Checkout ─> Success
      ├─> Search ───────────────^
      └─> PDP (из блоков витрины) ^
Header: Logo · Каталог · Поиск · Язык(RU/EN) · Валюта(₽/$) · Корзина
```
